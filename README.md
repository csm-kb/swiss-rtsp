## Getting Started

### Prerequisites

* npm
  ```sh
  npm install -g npm
  ```
  See [this link](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) for more info.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/csm-kb/swiss-rtsp.git
   ```
2. Install NPM packages
